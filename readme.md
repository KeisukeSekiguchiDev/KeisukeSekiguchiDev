### Quality Assurance

| 品質観点 | 実践内容 | 適用例 |
|---------|---------|--------|
| **自動テスト** | Playwright E2E / ユニットテスト | 85+テストケース、全パス |
| **コーディング規約** | ESLint / 命名規則 / 責務分離 | 1関数1責務、300行以内 |
| **ドキュメント** | 要件定義書 / 設計書 / API仕様 | 全工程でドキュメント作成 |
